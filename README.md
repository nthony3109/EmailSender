# Email Messaging Service App

## About

This is a simple **Spring Boot email messaging service** that sends emails to target users. It supports **HTML-formatted email bodies**, allowing rich and well-structured email content such as verification messages and notifications.

## Tools & Technologies

* **Java**
* **Spring Boot**
* **JavaMailSender**
* **MimeMessage** (for HTML email formatting)
* **Maven**
* **IntelliJ IDEA**

## Features

* Send emails to users via SMTP
* Support for HTML email body formatting
* Clean service-based architecture
* Easy configuration using `application.properties`

## Use Cases

* Email verification codes (OTP)
* User notifications
* System-generated emails
